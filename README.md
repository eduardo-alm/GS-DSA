# Monitor de Missão Espacial

Projeto desenvolvido para a Global Solution de Estruturas de Dados e Algoritmos.

## Objetivo

Monitorar informações básicas de uma missão espacial simulada.

## Funcionalidades

- Registro de temperatura
- Registro de energia
- Registro de comunicação
- Análise automática dos dados
- Alertas de superaquecimento
- Alertas de energia crítica
- Alertas de falha de comunicação
- Histórico de leituras
- Limpeza do histórico

## Tecnologias

- HTML
- CSS
- JavaScript

## Regras de Monitoramento

- Temperatura > 80°C → Alerta de superaquecimento
- Energia < 20% → Economia de energia
- Comunicação = 0 → Falha de comunicação
